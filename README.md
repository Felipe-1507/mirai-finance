## Sobre o projeto

O **MIRAI Finance** é uma aplicação web de controle financeiro pessoal focada em praticidade, organização mensal e uma visualização mais agradável das finanças do dia a dia.

O projeto nasceu como **Hylian Finance** e passou por um rebranding visual para **MIRAI Finance**, mantendo o backend Firebase original para preservar autenticação e dados já existentes.

A proposta do app é simples:

- organizar transações por mês
- acompanhar cartões, compras e mensalidades
- definir metas financeiras
- manter os dados sincronizados por usuário autenticado
- entregar uma interface rápida, direta e fácil de usar

## Principais funcionalidades

### Autenticação
- Login com **Google**
- Login com **e-mail e senha**
- Criação de conta
- Recuperação de senha
- Atualização de perfil do usuário

### Gestão financeira
- Resumo financeiro mensal
- Controle de **transações**
- Gestão de **cartões**
- Gestão de **compras**
- Controle de **mensalidades**
- Gestão de **metas financeiras**

### Experiência do usuário
- Interface em página única
- Loading inicial ao carregar dados
- Tema visual com suporte a modo escuro
- Ícones e gráficos para leitura rápida
- Exportação e importação de dados em **JSON**
